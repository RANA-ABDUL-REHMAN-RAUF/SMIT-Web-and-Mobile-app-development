### 1. HTML - Basic Website Structure (`01_HTML/`)

This assignment demonstrates the fundamentals of HTML5. It consists of a simple multi-page static website with basic layout, media elements, forms, and navigation.

- **Live Preview:** [assignment-smit-1](https://assignment-smit-1.surge.sh/index.html)
- **Pages Included:**
  - `index.html`: Home page with headings and lists.
  - `about.html`: About page.
  - `gallery.html`: Image gallery.

  - `login.html` & `register.html`: Form mockups.

### 2. CSS - Styling and Responsive Design (`02_CSS/`)

This section includes assignments focused on styling web pages with CSS, including layouts, responsive design, and modern CSS techniques.

- **Assignment 1: First Website Layout**

  - **Live Demo:** [smit-aman-assignment3](https://smit-aman-assignment3.surge.sh)
- **Assignment 2: PraRoz Course Website**
  - A static single-page design for a course website, styled with CSS and Font Awesome icons.
  - **Live Demo:** [Praroz](https://praroz-aman.surge.sh/)
- **Assignment 3: BR Architects Landing Page**
  - A fully responsive landing page for an architecture firm, built using the Bootstrap 5 framework.
  - **Live Demo:** [ BR Architects Landing Page](https://cerulean-bublanina-94b8c1.netlify.app/)

### 3. JavaScript - Interactivity (`03_JS/`)

This assignment introduces client-side scripting with JavaScript to create interactive and dynamic user experiences.

- **Assignment 1: JavaScript Fundamentals**
  - An interactive web app demonstrating core JavaScript concepts from chapters 1-7, including alerts, variables, data types, arithmetic operations, and basic DOM manipulation. The UI is built with Bootstrap.
  - **Live Demo:** [aman-js.netlify.app](https://aman-js.netlify.app/)

- **Assignment 2: JavaScript Conditionals & Prompts**
  - Explores conditional logic, prompts, and string concatenation (chapters 8-14) with interactive `if-else` examples.
  - **Live Demo:** [aman-js2.netlify.app](https://aman-js2.netlify.app/)

- **Assignment 3: JavaScript Arrays, Loops & DOM**
  - Focuses on arrays, loops, and DOM updates (chapters 15-20), including city list operations and table generation.
  - **Live Demo:** [aman-js3.netlify.app](https://aman-js3.netlify.app/)

- **Assignment 4: JavaScript Strings & Methods**
  - Covers common string methods and formatting (chapters 21-25) with Toastify-based validation and feedback.
  - **Live Demo:** [aman-js4.netlify.app](https://aman-js4.netlify.app/)

- **Assignment 5: Numbers, Integers & Decimals**
  - Practices numeric operations and `Math` methods (chapters 26-30): rounding, random values, dice, password generation, controlling decimal length, and GST calculation.
  - **Live Demo:** [aman-js5.netlify.app](https://aman-js5.netlify.app/)

- **Assignment 6: Date, Time & Functions**
  - Focuses on the JavaScript `Date` object, functions with parameters/return values, and intervals (chapters 31-38). Includes age calculation, birthday countdowns, and real-time clocks.
  - **Live Demo:** [aman-js6.netlify.app](https://aman-js6.netlify.app/)

## üöÄ Getting Started

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox).
- A code editor for viewing the source code (e.g., VS Code).

### How to Run

Since all projects are static front-end code, no special build process is required.

1.  **Clone or download** this repository.
2.  Navigate into any of the assignment folders (e.g., `01_HTML/`, `02_CSS/Assignment_1/`).
3.  Open the `index.html` file in your web browser to view the project.
4.  For a better development experience, you can run a simple local web server from the project's root directory:

    ```bash
    # If you have Python installed
    python -m http.server

    # If you have Node.js installed
    npx http-server
    ```

    Then, access the projects at `http://localhost:8000` (or the port specified by the server).
